OPCIÓN RECOMENDADA (La más fácil - 2 minutos):
═══════════════════════════════════════════════════════════════════════════════

1. Abre el archivo: SRS_PARA_PDF.html
   (Doble clic o click derecho → Abrir con → Google Chrome / Edge)

2. Presiona: Ctrl + P (o Archivo → Imprimir)

3. Configura:
   ✓ Destino: "Guardar como PDF"
   ✓ Tamaño: "Carta" o "A4"
   ✓ Márgenes: "Predeterminados"
   ✓ ✓ GRÁFICOS DE FONDO (IMPORTANTE para colores)

4. Click en "Guardar" → Elige ubicación → Listo!


═══════════════════════════════════════════════════════════════════════════════
OTRAS OPCIONES:
═══════════════════════════════════════════════════════════════════════════════

OPCIÓN B - Desde el HTML normal:
1. Abre: SRS_DOCUMENTACION.html
2. Ctrl + P
3. Configura igual que arriba
4. Guarda como PDF


OPCIÓN C - Usando Word:
1. Abre Microsoft Word
2. Archivo → Abrir → Selecciona SRS_COMPLETO.md
3. Ajusta formato (fuente Arial 12, interlineado 1.5)
4. Inserta logo del ITSC en portada
5. Agrega numeración de páginas
6. Archivo → Guardar como → PDF


OPCIÓN D - Usando VS Code:
1. Instala extensión: "Markdown PDF"
2. Abre SRS_COMPLETO.md en VS Code
3. F1 → "Markdown PDF: Export (pdf)"
4. Listo!

SOLUCIÓN DE PROBLEMAS:
═══════════════════════════════════════════════════════════════════════════════

PROBLEMA: El PDF sale sin colores
SOLUCIÓN: Activa "Gráficos de fondo" en configuración de impresión

PROBLEMA: Las tablas se cortan
SOLUCIÓN: En Word: Tabla → Propiedades → Fila → Desmarca "Dividir filas"

PROBLEMA: Archivo muy grande
SOLUCIÓN: Comprime en https://www.ilovepdf.com/compress_pdf

PROBLEMA: No aparece numeración
SOLUCIÓN: Insertar → Número de página → Pie de página → Centrado
